# poo-java
This repository contains Java code that exemplify concepts of Object Oriented Programming.

# Instruções
  1. Criar um diretório de trabalho [WorkDir]
  2. Entrar no [WorkDir] com o comando cd
  3. Usar o comando git clone https://github.com/ricdtaveira/poo-java.git
  4. Importar o diretório clonado para o Eclipse
